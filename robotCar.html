Ultrasonic Obstacle Avoidance Car

For my Microprocessor architecture class, I partnered with Dheeraj Nair to create an obstacle detecting car. Through our research and work we were able to succefully implement the components and logic required.
Project Paper | See it in action here!



The underside of the car, featuring the driver motor for the DC motor powered wheels

Overview

This project was built using the Texas Instruments TM4C. We chose this project due to our interest in smart cars. Drawing inspiration from manufacturers like Tesla, Rivian, and Lucid, we chose to create a simple car as a proof of concept to show that Ultrasonic sensors can be used to accurately gauge distance in autonomous automobiles.
