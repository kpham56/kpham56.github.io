NS3 Network Simulator

During my Summer 2021 internship with NASA Ames Research Center, I was tasked to create an AeroNautic Network Simulator.



The user can change these parameters to fit the desired scenario

Overview

This project was built using NS3, a free open source network event simulator. The goal was to create a configurable simulation in which users can adjust inputs to calculate the resulting throughput, packet delivery, and other metrics. While working with peers, this was mainly a solo project.


The resulting simulation opened in NetAnim
  • The Simulation
  • Used LTE topology to connect MME, SGW, ENBs, UEs, PGW
  • User definable
      - Amount of UE Nodes
      - Amount of ENB Nodes
      - Packet Size, Amount and time to send
      - Mobile or Stationary
  • Working Netanim xml output file to view simulation
  • Terminal and txt file output of simulation results
