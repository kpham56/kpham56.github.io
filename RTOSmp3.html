FreeRtos MP3 Player

For my Embedded design class, I partnered with Maurice Washington and Johnny Nguyen to create an MP3 player. The device features 4 mechanical switches, an lcd screen, and standalone usage using a powerbank. In order to interface each component, we had to read through numerous data sheets to create functions needed to communicate. This project used SPI and UART. We built the project over the span of 6 weeks.



The finished MP3 player

Overview

This project was built using the SJSU SJ2 board produced by LPC. As this is my RTOS project, I learned alot about design and control. Initially, we used CLI for basic testing over serial communication. As we implemented more functions, we were able to test the device only using the on board buttons and screen, just like a real consumer device. This project tied together everythin i've learned the last few years, through its use of data structures, timing, bitshifting, resource sharing and more.
