The Planck

Assembled a mechanical keyboard by hand. Opened 48 cherry MX clear switches, replaced with aftermarket springs to reduce acutation force, and lubricated all contact points to ensure quiet and consistent feedback. Solder switches to PCB, as well as an LED diode for each switch. Installed rubber O-rings on each keycap stem to silence chatter. Once assembled, the keyboard was programmed using QMK to change the keymaps. The Plank features multiple layers, an ultracompact size and an ergonomic ortholinear set up.
